How to Test the Frontend:

  1. Ensure all backend services are running (Auth, Profile, Match, Conversation, Venue, Voice, Safety). You can start them using their respective `npm run dev` commands in separate terminals.
  2. Verify the frontend is running on http://localhost:3000.
  3. Interact with the Profile tab to define your persona and interests.
  4. Check the Matches tab to see AI-generated matches based on your input.
  5. Explore the Chat and Connect tabs for more features.

If you encounter issues, check the console logs of each running service for errors.
